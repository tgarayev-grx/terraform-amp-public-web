The GRX Pay API lets you create invoices, manage payments, and integrate crypto payments into your application. All API requests use the base URL and require authentication.

**Next steps:**

- [Authentication](/pay/v1/authentication/) — create API keys and sign requests
- [How to call private endpoints](/pay/v1/how-to-call-private-endpoints/) — use signed requests for sensitive operations
- Browse the endpoints in the sidebar or open the [API reference](/pay/v1/api-reference/).
